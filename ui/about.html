About this project

What it is, how it's built, and what's under the hood.

Overview

my-f3-app is a demonstration application built on the Fat-Free Framework (F3). It pairs a clean Model–View–Controller layout with a JSON REST API and a server-rendered front end styled with Bootstrap 5.

How it's organised

  • config/ — INI files for app settings, database credentials and routes
  • app/Controllers/ — request handlers (web pages & API)
  • app/Models/ — data models on top of F3's SQL mapper
  • app/Support/ — the MySQL bootstrap that builds the schema
  • ui/ — Bootstrap templates rendered by F3's template engine
Tech stack
  • FrameworkFat-Free 3.9
  • LanguagePHP 8.3
  • DatabaseMySQL / MariaDB
  • Front endBootstrap 5
